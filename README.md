# *GitPal*

This project is developed in Python. The architecture we used is Django.  

-------------------------------------

**Library Dependencies**

-------------------------------------

Python 3.6.8

Numpy 1.16.4

Pandas 0.24.2

Networkx 2.3

-------------------------------------

**Run**

-------------------------------------

run code on local host

```shell
cd GitPal
python manage.py runserver 0.0.0.0
```
